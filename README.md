# 🤖 Chatbot Financiero – Versión Online (Web/API)

Este proyecto consiste en el desarrollo de un **Chatbot Financiero Inteligente**, diseñado para brindar información financiera exacta, actualizada y contextualizada de forma conversacional.
El chatbot permite consultar datos como inflación, dólar, acciones argentinas, criptomonedas, plazos fijos y riesgo país, combinando respuestas estructuradas con razonamiento mediante modelos de lenguaje (LLM).

## 🚀 Características
- Procesamiento de lenguaje natural (NLP)
- Datos financieros en tiempo real
- IA conversacional (LLM Groq)
- API REST
- Interfaz web

---

## ⚙️ Tecnologías utilizadas

- Backend:
Python y Flask

- Frontend:
HTML5, CSS3 y Javascript

- NLP & AI:
Groq (LLM)

- API:
Yahoo Finance y CoinGecko

- Base de datos:
MySQL y SQLAlchemy
